---

- `Ctrl+F` dentro de Elements busca etiquetas, texto y atributos.
- `Ctrl+Shift+F` busca across HTML, CSS y JS cargados.
- Uso de regex en búsquedas globales.

Ahorra tiempo localizando código relevante en grandes apps.
