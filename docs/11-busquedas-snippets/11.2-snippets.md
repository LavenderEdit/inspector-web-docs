En Sources > Snippets crea archivos `.js` cortos:

````js
// Ejemplo: log de todos los enlaces
[...document.querySelectorAll('a')].forEach(a => console.log(a.href));

Guarda y ejecuta en cualquier página con botón derecho → Run.

````
