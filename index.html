Empezar Guía

Inspecciona y modifica el DOM

Con el panel Elements puedes inspeccionar el árbol del DOM en tiempo real, modificar estilos CSS y ver cómo afecta inmediatamente al diseño de la página.

Depura JavaScript fácilmente

Usa el panel Sources para establecer breakpoints, observar el flujo del código y depurar errores directamente desde el navegador sin herramientas externas.

Optimiza el rendimiento web

El panel Performance te permite registrar, analizar y mejorar el rendimiento de carga, interacción y renderizado de tu sitio web.
